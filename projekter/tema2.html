Tema 2 - Grundlæggende web

I dette tema arbejdede jeg med at designe og kode et responsivt website fra bunden ved hjælp af HTML og CSS med fokus på struktur, mobile-first og semantisk korrekt kode.

Se projekt

Proces & læring

Her er de vigtigste elementer jeg arbejdede med i temaet - samlet og overskueligt.

Fokus

HTML, CSS, semantisk struktur, mobile-first og responsivt design.

Proces

Wireframes → HTML-struktur → CSS layout med Flexbox & Grid → responsiv tilpasning.

Designprincipper

Gestaltlove, typografi og visuelt hierarki for bedre brugervenlighed.

Teknik

HTML, CSS Grid, Flexbox og media queries.

Hvad har jeg lært?

I dette tema har jeg lært at opbygge et responsivt website fra bunden ved hjælp af HTML og CSS. Jeg har opnået forståelse for semantisk struktur, mobile-first-tankegang og grundlæggende designprincipper, som danner fundamentet for brugervenlige webløsninger.
